# Contributing to this Project

First off, thank you for considering contributing to this PixelPie Media project! It's people like you that make open source such a great community.

## Where do I go from here?
If you've noticed a bug or have a feature request, please make sure to check if there's already an open issue before creating a new one.

## Fork & pull request workflow
1. **Fork** the repo on GitHub
2. **Clone** the project to your own machine
3. **Commit** changes to your own branch
4. **Push** your work back up to your fork
5. Submit a **Pull Request** so that we can review your changes

## Code Style
Please adhere to the standard code style of the project. We use Prettier/ESLint for JavaScript/TypeScript and standard language formatters for others.
